# Bakgrunnsfjerner

Bakgrunnsfjerner er en intern kommunal webapp for opplasting av bilder, fremtidig bakgrunnsfjerning i nettleseren og nedlasting av resultatet som PNG.

## Stack

- React
- Vite
- Tailwind CSS
- Klientside ML via `@huggingface/transformers`
- Deploy til GitHub Pages

## Status

Dette er kun et grunnoppsett. Ingen ML-funksjonalitet er implementert ennå.
